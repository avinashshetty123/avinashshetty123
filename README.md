## 🔭 Featured Projects

### [TestIntegrity](https://github.com/avinashshetty123/testintegrity)
**AI-Powered Proctored Examination Platform**
- Cross-platform Electron app with real-time AI proctoring using YOLO and FaceNet
- Features face/eye tracking, multi-person detection, and integrity monitoring
- Integrated LiveKit for scalable audio/video sessions
- Python analysis pipeline for automated report generation

### [BloomBuddy](https://github.com/avinashshetty123/bloombuddy)
**AI Plant Identification and Care App**
- React Native mobile app with PlantNet API integration
- AI-powered plant care guidance using OpenAI models
- Location-aware suggestions based on weather and seasonal conditions
- Firebase backend for real-time data management

### [StargazeX](https://github.com/avinashshetty123/stargazex)
**Astronomy Platform**
- Location-based stargazing condition analysis
- Community event hosting system with RSVP management
- Telescope sharing network with scheduling capabilities
- Node.js/Express backend with predictive algorithms

### [MindMetrics](https://github.com/avinashshetty123/mindmetrics)
**Stress Detection Web Application**
- Real-time stress monitoring with Google Fit integration
- OAuth2 authentication and continuous data synchronization
- PSQ-based stress evaluation algorithms
- React + Tailwind CSS responsive interface
